====================================
tinyTiM - The tiny Topic Maps engine
====================================

What is tinyTiM CXTM?
---------------------
tinyTiM is a tiny Topic Maps engine which keeps topic maps in-memory.
This Topic Maps engine is meant to be used together with the TMAPI interfaces,
see <http://www.tmapi.org/> for details.
The CXTM subpackage provides the serialization of topic maps into Canonical XTM 
(CXTM) <http://www.isotopicmaps.org/cxtm/>.


Installation
------------
No real installation needed, simply put the ``tinytim-cxtm-<VERSION>.jar`` 
together with the ``tinytim-<VERSION>.jar`` and ``tmapi-1_0SP1.jar`` into your 
classpath.
